# dbApps10 Capstone Project - Grading Rubric

**Course:** Database Applications Development (145085)
**Medina County Career Center**

**Total Points: 120**

---

## Rubric

| Category | Criteria | Points |
|----------|----------|--------|
| **Design (30 pts)** | Topic description and business rules clearly defined | 5 |
| | ER diagram with entities, PKs, FKs, relationships labeled | 10 |
| | Data dictionary (all tables, columns, types, constraints) | 10 |
| | At least 3 tables with proper PK/FK relationships | 5 |
| **Build (30 pts)** | CREATE TABLE statements with constraints (NOT NULL, UNIQUE, DEFAULT, CHECK, FK) | 10 |
| | 20+ records per main table using INSERT statements | 10 |
| | Database runs without errors | 5 |
| | Tables verified with SELECT * | 5 |
| **Queries (30 pts)** | 10+ queries total | 5 |
| | Includes SELECT with WHERE | 3 |
| | Includes aggregate function (COUNT/SUM/AVG/MIN/MAX) | 3 |
| | Includes GROUP BY with HAVING | 4 |
| | Includes INNER JOIN | 5 |
| | Includes LEFT JOIN or multi-table JOIN | 5 |
| | Queries have comments explaining what they do | 5 |
| **Security & Documentation (15 pts)** | At least 1 parameterized query | 5 |
| | At least 1 transaction (BEGIN/COMMIT) | 5 |
| | Code comments throughout notebook | 5 |
| **Presentation (15 pts)** | Presented to class within 10 minutes | 5 |
| | Demonstrated 3-4 queries live | 5 |
| | Explained design choices | 5 |

---

## Submission Requirements

- Submit SQL script file with all CREATE TABLE, INSERT, and SELECT statements
- Include ER diagram (image or Lucidchart link)
- Include data dictionary document
- Present to class with live query demonstration
