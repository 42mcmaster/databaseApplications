## Огляд Концепцій

### Основи JupyterLab

JupyterLab — це місце, де ми пишемо весь код у цьому курсі. Ноутбуки дозволяють вам змішувати код та текст в одному файлі. Ви запускаєте клітинки коду, натискаючи **Shift + Enter**. Результати з'являються прямо під клітинкою. Клітинки Markdown дозволяють писати форматовані нотатки для документування вашої роботи.

### Змінні та Типи Даних

Змінні — це як ви зберігаєте інформацію в Python. Ви створюєте одну, вибираючи ім'я та призначаючи значення з `=`. Python автоматично визначає тип даних на основі того, що ви призначите. Чотири базові типи:

- **str** — текст, завжди в лапках: `"Alice"`
- **int** — цілі числа без десяткових: `17`
- **float** — числа з десяткової крапкою: `3.75`
- **bool** — `True` або `False` (з великої літери)

Використовуйте `type(variableName)` для перевірки типу змінної.

### print() та Коментарі

`print()` виводить значення на екран. Ви можете вивести текст, змінні або їх поєднання:

```python
print("Hello!")
print("Age:", studentAge)
```

Коментарі починаються з `#`. Python їх повністю ігнорує. Використовуйте коментарі для пояснення **чому** ваш код робить щось, а не просто **що** він робить.

### Базова Арифметика

Python підтримує стандартні математичні оператори: `+` (додавання), `-` (віднімання), `*` (множення), `/` (ділення), `**` (степень). Python дотримується нормального порядку операцій. Використовуйте дужки для контролю порядку, якщо потрібно.

```python
ticketPrice = 12.50
numTickets = 4
totalCost = ticketPrice * numTickets
```

### pandas та DataFrames

pandas — це Python бібліотека, яка спрощує роботу з табульованими даними (рядки та колонки). Ви імпортуєте її з `import pandas as pd`. **DataFrame** — це основна структура даних pandas — подумайте про неї як про електронну таблицю, завантажену в Python. Ви завантажуєте CSV файл з `pd.read_csv("filename.csv")`.

### Дослідження Даних

Після завантаження даних у DataFrame використовуйте ці методи для дослідження:

| Метод | Що він робить |
|-------|---------------|
| `.head()` | Показує перші 5 рядків |
| `.shape` | Повертає (кількість рядків, кількість колонок) |
| `.info()` | Показує назви колонок, типи даних та рахунки відсутніх значень |
| `.describe()` | Показує статистику для числових колонок |

### Контроль Версій

Контроль версій — це система, яка записує зміни файлів з часом, щоб ви могли повернутися до раніших версій, якщо потрібно. **Git** — це найпопулярніший інструмент контролю версій. **GitHub** — це сайт, який зберігає ваші Git проекти в Інтернеті. Наш робочий процес у класі: робота → збереження файлу → **commit** (снимок) → **push** (завантаження на GitHub).
