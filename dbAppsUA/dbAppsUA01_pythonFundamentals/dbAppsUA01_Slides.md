## Що таке JupyterLab?

**JupyterLab** — це інструмент для написання та запуску коду у веб-браузері.

- Файли називаються **ноутбуками (Notebooks)** (`.ipynb`)
- Код запускається невеликими частинами, які називаються **клітинками (Cells)**
- Результати виводяться прямо під кожною клітинкою
- Використовується аналітиками та розробниками у реальному світі

**Щоб створити ноутбук:** File → New → Notebook → Python 3

## Два Типи Клітинок

Коли ви відкриєте ноутбук, ви працюватимете з двома типами клітинок:

**Клітинка Коду (Code Cell)** — Запускає Python код
- На лівій стороні є `[ ]:`
- Натисніть **Shift + Enter** для запуску

**Клітинка Markdown (Markdown Cell)** — Відображає форматований текст та нотатки
- Без дужок
- Натисніть **Shift + Enter** для рендеринку

```python
# Це клітинка коду
print("Hello!")
```

**Порада:** Використовуйте клітинки Markdown для нотаток та пояснень у ваших ноутбуках.

## Що таке Змінна (Variable)?

**Змінна** — це контейнер, який зберігає значення, яке ви можете використовувати пізніше.

```python
studentName = "Alice"
```

Подумайте про це як про названу шухляду у картотеці. Ви кладете щось всередину, даєте їй позначку, і можете витягти це будь-коли.

## Чотири Базові Типи Даних

Python має чотири основні типи даних, які ви будете використовувати:

```python
studentName = "Alice"      # str   → текст
studentAge = 17            # int   → ціле число
studentGPA = 3.75          # float → число з десятковою крапкою
isEnrolled = True          # bool  → True або False
```

**Перевірити тип будь-чого:**
```python
type(studentName)   # <class 'str'>
type(studentAge)    # <class 'int'>
```

> **Ключова ідея:** Кожне значення в Python належить до одного з цих чотирьох типів. Знайте ваші типи!

## Математика в Python

Python працює як калькулятор. Ось основні операції:

| Оператор | Значення | Приклад |
|----------|---------|---------|
| `+` | Додавання | `5 + 3` → `8` |
| `-` | Віднімання | `10 - 4` → `6` |
| `*` | Множення | `3 * 7` → `21` |
| `/` | Ділення | `15 / 4` → `3.75` |
| `**` | Степень | `2 ** 3` → `8` |

**Приклад з реального світу:**
```python
ticketPrice = 12.50
numTickets = 4
totalCost = ticketPrice * numTickets
print(totalCost)    # Виводить: 50.0
```

Python обчислює вираз, зберігає відповідь у `totalCost`, і ви можете її використовувати пізніше.

## Що таке pandas?

**pandas** — це бібліотека для роботи з даними у таблицях (як у Excel).

Коли ви завантажуєте CSV-файл у pandas, він стає **DataFrame** — примхливою таблицею, яку ви можете аналізувати.

```python
import pandas as pd
titanic = pd.read_csv("Titanic Dataset.csv")
```

> **pandas = Професійний Інструмент Для Роботи З Даними**

## Дослідження Даних За Допомогою pandas

Один раз завантаживши дані, ви можете дослідити їх простими командами:

```python
titanic.head()       # Показати перші 5 рядків
titanic.shape        # Скільки рядків та колонок?
titanic.info()       # Назви колонок та типи даних
titanic.describe()   # Статистика: середнє значення, мінімум, максимум
```

Ці команди допомагають вам зрозуміти, які дані у вас є, перш ніж їх аналізувати.

## Контроль Версій (Version Control)

**Контроль версій** відслідковує кожну зміну, яку ви робите у своїх файлах з часом.

**Git** — це інструмент. **GitHub** — це місце, де ваші файли зберігаються в Інтернеті.

| Термін | Значення |
|--------|---------|
| **Repository (Repo)** | Ваша папка проекту, відстежувана Git |
| **Commit** | Збережена снимок вашої роботи |
| **Push** | Завантажити ваші commits на GitHub |
| **Pull** | Завантажити зміни з GitHub |

**Наш робочий процес:** Пишете код → Зберігаєте файл → Commit → Push на GitHub
