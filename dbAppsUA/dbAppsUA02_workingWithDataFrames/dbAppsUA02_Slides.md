## Що таке DataFrame?

**DataFrame** — це таблиця даних з рядками та стовпцями, як електронна таблиця у Python.

Уявіть це як ваші дані, організовані в акуратну сітку.

```python
import pandas as pd
titanic = pd.read_csv("Titanic Dataset.csv")
```

> **DataFrame** — це таблиця з позначеними рядками і стовпцями, яку можна читати, фільтрувати та розраховувати.

titanic.head()       # Показати перших 5 рядків

## Отримання статистики

Pandas може швидко розраховувати базову математику для стовпців:

```python
titanic["age"].mean()      # Середній вік
titanic["fare"].max()      # Найвища плата за квиток
titanic["survived"].sum()  # Загальна кількість тих, хто вижив
```

> **У SQL** ви б використали `AVG()`, `MAX()`, `SUM()` — та ж ідея!

---

## Вибір одного стовпця

Щоб отримати один стовпець, використовуйте **одинарні дужки**:

```python
names = titanic["name"]
```

Це дає вам **Series** (Серію) — один стовпець даних.

---

## Вибір кількох стовпців

Щоб вибрати кілька стовпців, використовуйте **подвійні дужки**:

```python
subset = titanic[["name", "age", "survived"]]
```

Це дає вам **DataFrame** — нову таблицю лише з цими стовпцями.

> **У SQL:** `SELECT name, age, survived FROM titanic`

---

## Фільтрування рядків

Ви можете залишити тільки рядки, які відповідають умові:

```python
adults = titanic[titanic["age"] > 18]
firstClass = titanic[titanic["pclass"] == 1]
women = titanic[titanic["sex"] == "female"]
```

Читайте це як: «Показати мені рядки, де вік більший за 18» або «Показати мені пасажирів першого класу».

> **У SQL:** `WHERE pclass = 1` або `WHERE sex = 'female'`

---

## Комбінування умов

Використовуйте `&` для AND (і) та `|` для OR (або). **Важливо: завжди використовуйте дужки!**

```python
femaleSurvivors = titanic[
    (titanic["sex"] == "female") & (titanic["survived"] == 1)
]
```

Це знаходить рядки, де стать «female» **І** survived дорівнює 1.

> **У SQL:** `WHERE sex = 'female' AND survived = 1`

---

## Сортування даних

Сортування в порядку зростання (від меншого до більшого):

```python
byAge = titanic.sort_values("age")
```

Сортування в порядку спадання (від більшого до меншого):

```python
byFare = titanic.sort_values("fare", ascending=False)
```

> **У SQL:** `ORDER BY age` або `ORDER BY fare DESC`

---

## Підрахунок значень

`.value_counts()` підраховує, скільки разів кожне значення з'являється у стовпці:

```python
titanic["pclass"].value_counts()
```

Результат:
```
3    709
1    323
2    277
```

Це говорить нам: 709 пасажирів були в 3-му класі, 323 у 1-му класі тощо.

> **У SQL:** `SELECT pclass, COUNT(*) FROM titanic GROUP BY pclass`

---

## Pandas → SQL еквіваленти

| Що ви хочете | Код pandas | SQL еквівалент |
|---|---|---|
| Вибрати стовпці | `df[["col1", "col2"]]` | `SELECT col1, col2` |
| Фільтрувати рядки | `df[df["col"] > 5]` | `WHERE col > 5` |
| Сортувати результати | `df.sort_values("col")` | `ORDER BY col` |
| Розрахувати середнє | `df["col"].mean()` | `AVG(col)` |
| Підрахувати | `df["col"].value_counts()` | `GROUP BY col` |

## Ключові висновки

- **DataFrames** — це таблиці даних, як електронні таблиці у Python
- **Вибирайте стовпці** за допомогою дужок: одинарні `[]` для одного, подвійні `[[]]` для багатьох
- **Фільтруйте рядки**, записуючи умови: `df[df["age"] > 30]`
- **Розраховуйте статистику** за допомогою `.mean()`, `.sum()`, `.max()` тощо
- **Кожна операція pandas** матиме еквівалент у SQL — ви побачите його на уроці 03!
