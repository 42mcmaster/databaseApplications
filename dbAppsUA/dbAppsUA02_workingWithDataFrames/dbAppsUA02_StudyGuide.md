## Огляд концепцій

### Що таке DataFrame?

DataFrame — це основний спосіб, як pandas зберігає дані. Уявіть це як електронну таблицю: рядки — це окремі записи (як пасажир), а стовпці — це атрибути (як ім'я, вік, тариф). Коли ви завантажуєте CSV за допомогою `pd.read_csv()`, результатом є DataFrame.

### Вибір стовпців

Щоб отримати один стовпець, використовуйте одинарні дужки: `titanic["name"]`. Це повертає **Series** (один стовпець). Щоб отримати кілька стовпців, використовуйте подвійні дужки зі списком назв: `titanic[["name", "age", "survived"]]`. Це повертає менший DataFrame. Назви стовпців чутливі до регістру.

### Фільтрування рядків

Фільтрування означає залишення тільки рядків, де умова істинна. Схема: `df[condition]`. Наприклад, `titanic[titanic["age"] > 30]` залишає тільки пасажирів старше 30 років. Умова `titanic["age"] > 30` створює значення True/False для кожного рядка, і зовнішні дужки залишають тільки True рядки.

Щоб комбінувати умови, використовуйте `&` для «і» та `|` для «або». Кожна умова повинна бути в своїх дужках: `titanic[(titanic["age"] > 30) & (titanic["sex"] == "female")]`.

### Сортування

`.sort_values("columnName")` сортує рядки за цим стовпцем. Зростання (від низького до високого) — це за замовчуванням. Використовуйте `ascending=False` для спадання (від високого до низького). Сортування не змінює оригінальний DataFrame — воно повертає нову відсортовану копію.

### Статистика

Pandas надає методи для розрахування статистики на числових стовпцях. `.mean()` дає середнє значення, `.max()` та `.min()` дають екстремуми, `.sum()` додає все разом, а `.count()` говорить вам, скільки існує не відсутніх значень. Ви можете комбінувати це з фільтруванням, щоб отримати статистику для підмножини: `titanic[titanic["pclass"] == 1]["fare"].mean()` дає середній тариф тільки для пасажирів першого класу.

### Value Counts

`.value_counts()` — це швидкий спосіб побачити, як розподілені дані. Він підраховує, скільки разів кожне унікальне значення з'являється, і сортує за частотою. Це корисно для категоріальних стовпців, таких як `pclass`, `sex` або `survived`.

### Міст Pandas-to-SQL

Кожна операція, яку ви вивчаєте в pandas, має прямий еквівалент у SQL. Це навмисне — розуміння маніпуляції даних у pandas значно полегшує навчання SQL. Таблиця нижче показує відображення.

---

## Pandas → SQL Довідка

| Що ви хочете зробити | Pandas | SQL |
|---------------------|--------|-----|
| Вибрати конкретні стовпці | `df[["name", "age"]]` | `SELECT name, age FROM table` |
| Отримати всі стовпці | `df` | `SELECT * FROM table` |
| Фільтрувати рядки (рівно) | `df[df["pclass"] == 1]` | `WHERE pclass = 1` |
| Фільтрувати рядки (більше, ніж) | `df[df["age"] > 30]` | `WHERE age > 30` |
| Комбінувати з AND | `df[(cond1) & (cond2)]` | `WHERE cond1 AND cond2` |
| Комбінувати з OR | `df[(cond1) \| (cond2)]` | `WHERE cond1 OR cond2` |
| Сортування за зростанням | `df.sort_values("age")` | `ORDER BY age` |
| Сортування за спаданням | `df.sort_values("age", ascending=False)` | `ORDER BY age DESC` |
| Середнє значення | `df["age"].mean()` | `SELECT AVG(age)` |
| Максимум | `df["fare"].max()` | `SELECT MAX(fare)` |
| Мінімум | `df["age"].min()` | `SELECT MIN(age)` |
| Сума | `df["survived"].sum()` | `SELECT SUM(survived)` |
| Кількість | `df["age"].count()` | `SELECT COUNT(age)` |
| Кількість за категоріями | `df["pclass"].value_counts()` | `SELECT pclass, COUNT(*) GROUP BY pclass` |

## Карточка швидкої довідки

### Завантаження даних
```python
import pandas as pd
titanic = pd.read_csv("Titanic Dataset.csv")
```

### Вибір стовпців
```python
titanic["name"]                          # Один стовпець (Series)
titanic[["name", "age", "survived"]]     # Кілька стовпців (DataFrame)
```

### Фільтрування
```python
titanic[titanic["age"] > 30]                                        # Одна умова
titanic[(titanic["sex"] == "female") & (titanic["survived"] == 1)]  # AND
titanic[(titanic["pclass"] == 1) | (titanic["pclass"] == 2)]       # OR
```

### Сортування
```python
titanic.sort_values("age")                    # Зростання (за замовчуванням)
titanic.sort_values("fare", ascending=False)  # Спадання
```

### Статистика
```python
titanic["age"].mean()       # Середнє значення
titanic["fare"].max()       # Максимум
titanic["fare"].min()       # Мінімум
titanic["survived"].sum()   # Сума
titanic["age"].count()      # Кількість (не відсутні)
```

### Value Counts
```python
titanic["pclass"].value_counts()    # Підрахунок за категоріями
titanic["sex"].value_counts()       # Підрахунок за категоріями
```
