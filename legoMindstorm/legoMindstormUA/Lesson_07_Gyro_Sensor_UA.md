## Теорія – Що таке гіроскопічний датчик?

Гіроскопічний датчик вимірює **обертання** — як сильно робот крутиться та як швидко. Його не цікавлять відстань, світло чи дотик. Його цікавить тільки скручування.

```
Гіроскопічний датчик (вид зверху)
       ↑ 0°
       │
  -90° ─┼─ +90°
       │
      180°
```

**Ключові факти:**
- Вимірює **загальний кут** обертання робота в градусах
- Також вимірює **швидкість обертання** в градусах на секунду
- Обертання за годинниковою стрілкою — позитивні, проти годинникової стрілки — негативні
- Підключається до будь-якого порту датчика (S1–S4)
- **Увага щодо дрейфу:** датчик повинен залишатися абсолютно нерухомим при першому включенні робота. Якщо ви рухаєте його під час запуску, показання повільно змінюватимуться навіть коли робот нерухомий. Завжди скидайте кут перед поворотом, на якому вам це важливо.

## Частина 1 – Налаштування та перше читання

Підключіть гіроскопічний датчик до **порту 2** на вашому EV3 кирпічу. Встановіть його плоско на верхній частині вашої мобільної бази так, щоб стрілки на датчику були спрямовані вгору (не збоку). **Тримайте робота абсолютно нерухомо** під час підключення та запуску програми — це запобігає дрейфу.

Створіть новий проект під назвою `gyro_test`. Введіть цей код:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import GyroSensor
from pybricks.parameters import Port
from pybricks.tools import wait

ev3 = EV3Brick()

# Створіть об'єкт гіроскопічного датчика на порту 2.
gyro = GyroSensor(Port.S2)

# Скидайте кут на 0 перед тим, як починати читання.
gyro.reset_angle(0)

ev3.speaker.beep()
ev3.screen.clear()

# Читайте кут 20 разів, двічі на секунду.
for i in range(20):
    angle = gyro.angle()
    ev3.screen.print("Angle: " + str(angle))
    wait(500)
```

Запустіть його, потім підніміть робота та повільно обертайте його ліворуч та праворуч. Спостерігайте, як змінюється кут — поворот праворуч додає до числа, поворот ліворуч віднімає.

**Нові команди:**

| Команда | Що вона робить |
|---|---|
| `GyroSensor(Port.S2)` | Створює об'єкт гіроскопічного датчика на порту 2 |
| `gyro.reset_angle(0)` | Встановлює поточний кут на 0 (використовуйте перед кожним вимірюваним поворотом) |
| `gyro.angle()` | Повертає загальне обертання в градусах (позитивне = праворуч, негативне = ліворуч) |
| `gyro.speed()` | Повертає поточну швидкість обертання в градусах на секунду |

## Частина 2 – Реагування на обертання

Гіроскоп ідеальний для **точних поворотів**. Ось приклад, який повертає робота рівно на 90 градусів праворуч, незалежно від того, наскільки швидко він крутиться:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor, GyroSensor
from pybricks.parameters import Port
from pybricks.robotics import DriveBase
from pybricks.tools import wait

ev3 = EV3Brick()
left_motor  = Motor(Port.B)
right_motor = Motor(Port.C)
robot = DriveBase(left_motor, right_motor, wheel_diameter=55.5, axle_track=104)

gyro = GyroSensor(Port.S2)
gyro.reset_angle(0)   # почніть з нуля

ev3.speaker.beep()

# Крутіться на місці, поки ми не повернемося на 90 градусів праворуч.
while True:
    if gyro.angle() >= 90:
        robot.stop()
        break
    else:
        robot.drive(0, 60)   # швидкість 0, швидкість повороту 60 град/с
    wait(10)

ev3.screen.print("Turned 90 degrees")
ev3.screen.print("Final: " + str(gyro.angle()))
```

Ось другий приклад — їхати вперед, перевіряючи, що робот не зійшов з курсу. Якщо він повернеться більш ніж на 5 градусів в будь-яку сторону, зупиніться:

```python
gyro.reset_angle(0)

while True:
    angle = gyro.angle()

    if angle > 5 or angle < -5:
        robot.stop()
        ev3.speaker.beep(880, 300)
        break
    else:
        robot.drive(150, 0)
    wait(10)

ev3.screen.print("Off course!")
ev3.screen.print("Angle: " + str(angle))
```

**Той самий основний паттерн датчика, як у інших уроках:**
```
while True:
    # читайте датчик
    if умова:
        # реагуйте
    else:
        # продовжуйте
    wait(10)
```

## Частина 3 – Ваш проект

Розробіть робота, який використовує **гіроскопічний датчик** та **два мотори** для досягнення мети, яку **ви вибираєте**.

### Правила

1. Ваш робот повинен використовувати **гіроскопічний датчик** для прийняття рішень
2. Ваш робот повинен використовувати **DriveBase** (два мотори) для руху
3. Ви повинні використовувати **`while` цикл** з показаннями датчика всередині нього
4. Ваша програма повинна **скинути кут гіроскопа** перед першим вимірюваним поворотом
5. Ви обираєте, яка **мета** вашого робота — будьте креативні

### Деякі ідеї для початку

- **Драйвер квадрата** — їхати вперед на встановлену відстань, повернути рівно на 90 градусів, повторити чотири рази, щоб накреслити квадрат
- **Драйвер трикутника або п'ятикутника** — та сама ідея, але з іншими кутами повороту
- **Компасний робот** — обернути робота в напрямку, потім завжди повертатися до цього заголовка, незалежно від того, як ви його крутите
- **Лічильник обертань** — пищати та відображати на екрані лічильник щоразу, коли робот завершує повний оборот на 360 градусів
- **Детектор дрейфу** — їхати прямо, і якщо кут дрейфує більш ніж на 5 градусів, автоматично виправити назад на 0
- **Поворот на команду** — їхати вперед, поки кут не досягне цільового значення, потім зупинитися

Гіроскоп не ідеальний. Якщо робот крутиться занадто швидко, він може перевищити цільовий кут перед тим, як код знову перевірить. Якщо він крутиться занадто повільно, він може зупинитися раніше.
