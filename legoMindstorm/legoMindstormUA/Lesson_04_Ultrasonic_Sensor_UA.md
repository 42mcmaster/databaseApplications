## Загальна картина – Що таке ультразвуковий датчик?

Ультразвуковий датчик працює як ехолокація кажана. Він надсилає імпульс звуку високої частоти та вимірює, скільки часу потрібно для відбиття назад. З цього він обчислює **відстань до найближчого об'єкта** попереду.

```
Ультразвуковий датчик          Об'єкт
   )))  →  →  →  →  →          |
   (((  ←  ←  ←  ←  ←          |
                                |
   "Той об'єкт знаходиться на відстані 35 см."
```

**Ключові факти:**
- Вимірює відстань від **0 до 255 см** (приблизно 8 футів)
- За замовчуванням повертає відстань у **міліметрах**
- Підключається до одного з **портів датчиків** на нижній частині EV3 brick (Порти 1, 2, 3 або 4)
- Найкраще працює, коли направлений безпосередньо на пласку поверхню

## Частина 1 – Налаштування та перше зчитування

Підключіть ультразвуковий датчик до **порту 4** на вашому EV3 brick. Встановіть його на передній частині вашої конструкції так, щоб він дивився вперед.

Створіть новий проєкт із назвою `ultrasonic_test`. Введіть цей код:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor, UltrasonicSensor
from pybricks.parameters import Port
from pybricks.tools import wait

ev3 = EV3Brick()

# Створюємо об'єкт ультразвукового датчика на порту 4.
obstacle_sensor = UltrasonicSensor(Port.S4)

ev3.speaker.beep()
ev3.screen.clear()

# Зчитуємо відстань 10 разів, по одному разу на секунду.
for i in range(10):
    distance = obstacle_sensor.distance()
    ev3.screen.print("Dist: " + str(distance) + " mm")
    wait(1000)
```

Запустіть, а потім переміщуйте руку ближче та далі від датчика під час виконання. Спостерігайте, як числа змінюються на екрані EV3.

**Нові команди:**

| Команда | Що робить |
|---|---|
| `UltrasonicSensor(Port.S4)` | Створює об'єкт датчика на порту 4 (порти датчиків використовують `S1`–`S4`) |
| `obstacle_sensor.distance()` | Повертає відстань до найближчого об'єкта в мм |

## Частина 2 – Реакція на відстань

Тепер змусимо робота **щось робити** залежно від того, що зчитує датчик. Ось де `while True` та `if/else` поєднуються з `robot.drive()`.

Уважно прочитайте цей приклад:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor, UltrasonicSensor
from pybricks.parameters import Port
from pybricks.robotics import DriveBase
from pybricks.tools import wait

ev3 = EV3Brick()
left_motor  = Motor(Port.B)
right_motor = Motor(Port.C)
robot = DriveBase(left_motor, right_motor, wheel_diameter=55.5, axle_track=104)

obstacle_sensor = UltrasonicSensor(Port.S4)

ev3.speaker.beep()

# Їдемо вперед, поки щось не опиниться ближче ніж 200 мм (приблизно 8 дюймів).
while True:
    distance = obstacle_sensor.distance()

    if distance < 200:
        robot.stop()
        ev3.speaker.beep(880, 200)
        break
    else:
        robot.drive(150, 0)

    wait(10)

ev3.screen.print("Object detected!")
ev3.screen.print("Dist: " + str(distance) + " mm")
```

**Що відбувається:**
- `while True:` — цикл виконується нескінченно, поки не буде викликано `break`
- Кожна ітерація циклу: зчитуємо датчик, вирішуємо, що робити
- `robot.drive(150, 0)` — їдемо вперед зі швидкістю 150 мм/с, без повороту (пам'ятайте з Уроку 03: `drive()` не блокує, тому ми можемо продовжувати перевіряти датчик)
- `if distance < 200:` — якщо щось знаходиться в межах 200 мм, зупиняємось і виходимо з циклу
- `wait(10)` — маленька пауза, щоб не перевантажувати brick

**Це основний шаблон для програм на основі датчиків:**
```
while True:
    зчитати датчик
    if умова:
        реагувати
    else:
        продовжувати
    wait(10)
```

Ви будете використовувати цей шаблон у кожному уроці з датчиками відтепер.

## Частина 3 – Ваш проєкт

Тепер ваша черга. Ви розробите робота, який використовує **ультразвуковий датчик** та **два мотори** для досягнення цілі, яку **ви обираєте**.

### Правила

1. Ваш робот повинен використовувати **ультразвуковий датчик** для прийняття рішень
2. Ваш робот повинен використовувати **DriveBase** (два мотори) для руху
3. Ви повинні використовувати **цикл `while`** із зчитуванням датчика всередині
4. Ви самі вирішуєте, яка **мета** вашого робота — будьте творчими

### Кілька ідей для початку

Виберіть одну з них або придумайте свою:
- **Робот-охоронець** — їде вперед і назад, патрулюючи, і повертається, коли виявляє стіну
- **Сором'язливий робот** — відступає назад, коли щось наближається надто близько
- **Помічник паркування** — повільно їде вперед і подає частіші сигнали, коли наближається до об'єкта (як датчик паркування автомобіля)
- **Дослідник** — їде вперед, і коли наближається до стіни, повертає на випадковий кут і продовжує
- **Охоронна сигналізація** — стоїть на місці, але подає сигнал тривоги та блимає індикатором, коли щось входить у його зону
- **Слідуй за лідером** — намагається триматися на фіксованій відстані за вашою рукою, поки ви її переміщуєте
