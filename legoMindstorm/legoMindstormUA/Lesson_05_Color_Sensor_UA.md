## Інформація – що таке датчик кольору?

Датчик кольору світить світлом і вимірює те, що повертається. У нього є три режими:

**Режим кольору** — виявляє один із 7 кольорів: чорний, синій, зелений, жовтий, червоний, білий, коричневий (плюс «без кольору», якщо не може визначити). Найкраще працює на відстані 1–2 см від поверхні.

**Режим відбитого світла** — вимірює, скільки світла відбивається від поверхні. Повертає число від 0 (дуже темно) до 100 (дуже яскраво). Відмінно підходить для виявлення ліній на підлозі.

**Режим навколишнього світла** — вимірює, скільки світла в кімнаті (без випромінювання власного світла). Повертає 0–100.

```
Датчик кольору
   ↓ світить світлом вниз
   ═══════════════
   темна поверхня  →  низьке значення відбитого світла (10-20)
   світла поверхня →  високе значення відбитого світла (70-90)
```

## Частина 1 – налаштування та перше читання

Підключіть датчик кольору до **порту 3** на цеглі EV3. Монтуйте його на нижній передній частині вашої бази робота, спрямований **вниз** на підлогу, приблизно на 1–2 см над поверхнею.

Створіть новий проект під назвою `color_test`. Введіть цей код:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor, ColorSensor
from pybricks.parameters import Port, Color
from pybricks.tools import wait

ev3 = EV3Brick()

# Створіть об'єкт датчика кольору на порту 3.
color_sensor = ColorSensor(Port.S3)

ev3.speaker.beep()
ev3.screen.clear()

# Прочитайте колір і світло 10 разів, один раз на секунду.
for i in range(10):
    color = color_sensor.color()
    light = color_sensor.reflection()

    ev3.screen.print("C:" + str(color) + " L:" + str(light))
    wait(1000)
```

Запустіть програму і рухайте робота над різними поверхнями — темний стіл, білий папір, кольоровий папір. Спостерігайте, як змінюються значення.

**Нові команди:**

| Команда | Що вона робить |
|---|---|
| `ColorSensor(Port.S3)` | Створює об'єкт датчика кольору на порту 3 |
| `color_sensor.color()` | Повертає виявлений колір (наприклад, `Color.BLACK`, `Color.WHITE`) |
| `color_sensor.reflection()` | Повертає інтенсивність відбитого світла (0 = темно, 100 = яскраво) |
| `color_sensor.ambient()` | Повертає рівень навколишнього світла (0 = темно, 100 = яскраво) |

**Значення кольорів, які можна перевіряти:**
`Color.BLACK`, `Color.BLUE`, `Color.GREEN`, `Color.YELLOW`, `Color.RED`, `Color.WHITE`, `Color.BROWN`, `None` (колір не виявлено)

## Частина 2 – реагування на колір і світло

Ось приклад, який рухається вперед і зупиняється, коли бачить темну лінію (як чорна стрічка на світлій підлозі):

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor, ColorSensor
from pybricks.parameters import Port, Color
from pybricks.robotics import DriveBase
from pybricks.tools import wait

ev3 = EV3Brick()
left_motor  = Motor(Port.B)
right_motor = Motor(Port.C)
robot = DriveBase(left_motor, right_motor, wheel_diameter=55.5, axle_track=104)

color_sensor = ColorSensor(Port.S3)

ev3.speaker.beep()

# Рухайтеся вперед, поки датчик не побачить темну поверхню.
while True:
    light = color_sensor.reflection()

    if light < 20:
        robot.stop()
        ev3.speaker.beep(880, 300)
        break
    else:
        robot.drive(100, 0)

    wait(10)

ev3.screen.print("Dark line found!")
ev3.screen.print("Light: " + str(light))
```

Ось приклад, який реагує на **конкретні кольори**:

```python
# Усередину циклу while:
color = color_sensor.color()

if color == Color.RED:
    robot.stop()
    ev3.speaker.beep(440, 500)
    ev3.screen.print("RED - Stop!")

elif color == Color.GREEN:
    robot.drive(200, 0)
    ev3.screen.print("GREEN - Go!")

elif color == Color.BLUE:
    robot.turn(90)
    ev3.screen.print("BLUE - Turn!")
```

**Той же базовий шаблон, що й урок 04:**
```
while True:
    прочитайте датчик
    if умова:
        реагуйте
    else:
        продовжуйте
    wait(10)
```

### Правила

1. Ваш робот повинен використовувати **датчик кольору** для прийняття рішень (режим кольору, режим відбитого світла або обидва)

3. Ваш робот повинен використовувати **DriveBase** (два мотора) для руху
4. Ви повинні використовувати цикл **`while`** з читанням датчика усередину нього
5. Ви самі вирішуєте, яка **мета** вашого робота — будьте креативні

### Деякі ідеї для початку

Виберіть одну з них або придумайте свою:
- **Послідовник лінії** — слідує за краєм чорної лінії на білій поверхні, використовуючи значення відбитого світла
- **Сортер за кольором** — рухається через кольорові зони і виконує різні дії для кожного кольору (звук, поворот, обертання, зупинка)
- **Детектор краю** — рухається на столі і повертається назад, перш ніж упасти з краю
- **Робот світлофора** — рухається, поки не побачить червоне (зупинка), чекає на зелене (йти), повертається на жовтому
- **Бот патруля** — рухається туди-сюди між двома лініями кольорової стрічки
- **Комбо: датчик кольору + датчик відстані коридорний бот** — рухається вперед, поки датчик відстані не побачить стіну, потім використовує датчик кольору для вибору напрямку повороту на основі кольорової стрічки на підлозі
