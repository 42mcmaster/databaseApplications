## Загальна картина – Мотори та порти

Ваш EV3 brick має **чотири вихідні порти** з позначками **A, B, C, D** зверху. Мотори підключаються до цих портів. У коді ми вказуємо Python, до якого порту підключений мотор, щоб він знав, куди надсилати команди.

```
EV3 Brick (вигляд зверху)
 ┌──────────────────────┐
 │  [A]  [B]  [C]  [D]  │  ← Порти для моторів
 │                       │
 │       EV3 Brick       │
 └──────────────────────┘
```

Для цього уроку підключіть **великий мотор** до **порту B**.

---

## Частина 1 – Перша програма з мотором

Створіть новий проєкт із назвою `motors`. Замініть усе у `main.py` наступним кодом:

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor
from pybricks.parameters import Port
from pybricks.tools import wait

# Створюємо об'єкти brick і мотора.
ev3 = EV3Brick()
motor = Motor(Port.B)

# Сигналізуємо про готовність.
ev3.speaker.beep()

# Обертаємо мотор зі швидкістю 360 градусів за секунду протягом 2 секунд.
motor.run_time(360, 2000)

ev3.screen.print("Done!")
```

Запустіть за допомогою **F5**. Мотор повинен обертатися 2 секунди, а потім зупинитися.

## Частина 2 – Команди для мотора

Є чотири основних способи змусити мотор рухатися. Кожен корисний у різних ситуаціях.

### `motor.run_time(speed, time)`

Запускає мотор із заданою швидкістю протягом певного часу, потім зупиняється.

```python
# Обертатися зі швидкістю 360 гр./с протягом 3 секунд.
motor.run_time(360, 3000)

# Обертатися назад зі швидкістю 180 гр./с протягом 1,5 секунди.
motor.run_time(-180, 1500)
```

- **speed** — у градусах за секунду. 360 = один повний оберт за секунду.
- **time** — у мілісекундах. 1000 мс = 1 секунда.
- Від'ємна швидкість = протилежний напрямок.
- Ця команда **блокує** — програма чекає тут, поки час не закінчиться.

---

### `motor.run_angle(speed, angle)`

Обертає мотор на задану кількість градусів, потім зупиняється.

```python
# Повернути рівно на 360 градусів (один повний оберт).
motor.run_angle(360, 360)

# Повернути на 90 градусів (чверть оберту).
motor.run_angle(360, 90)

# Повернути назад на 180 градусів.
motor.run_angle(360, -180)
```

- **speed** — як швидко (градусів за секунду).
- **angle** — як далеко (градусів обертання). Від'ємне = протилежний напрямок.
- Також блокує до завершення руху.

---

### `motor.run_target(speed, target_angle)`

Обертає мотор до певної позиції (абсолютного кута), як стрілки годинника.

```python
# Перейти в позицію 90 градусів.
motor.run_target(360, 90)
wait(1000)

# Перейти в позицію 270 градусів.
motor.run_target(360, 270)
wait(1000)

# Повернутися до 0 (початкова позиція).
motor.run_target(360, 0)
```

- **target_angle** — це позиція, а не відстань. Виклик `run_target(360, 90)` двічі поспіль нічого не зробить вдруге — мотор вже знаходиться в позиції 90.
- Це відрізняється від `run_angle()`, де кожен виклик переміщує мотор *далі* на задану кількість.

---

### `motor.run(speed)`

Запускає мотор і **продовжує його обертати**, поки ви не викличете `motor.stop()`. На відміну від інших команд, ця не блокує — наступний рядок коду виконується одразу.

```python
# Починаємо обертання.
motor.run(360)

# Мотор працює, поки ми робимо інші речі.
wait(3000)

# Зупиняємо мотор.
motor.stop()
```

Це корисно, коли ви хочете, щоб мотор працював під час перевірки кнопок або датчиків — саме це ми й робитимемо далі.

## Частина 3 – Використання циклу

Цикл `for` повторює блок коду. Це корисно, коли ви хочете, щоб мотор виконував щось кілька разів.

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor
from pybricks.parameters import Port
from pybricks.tools import wait

ev3 = EV3Brick()
motor = Motor(Port.B)

ev3.speaker.beep()

# Обертаємося вперед і назад 3 рази.
for i in range(3):
    ev3.screen.print("Lap " + str(i + 1))
    motor.run_angle(360, 360)     # Вперед на один повний оберт
    wait(300)
    motor.run_angle(360, -360)    # Назад на один повний оберт
    wait(300)

ev3.screen.print("Done!")
ev3.speaker.beep(880, 300)
```

Пам'ятайте:
- `range(3)` запускає цикл 3 рази (i = 0, 1, 2).
- `str(i + 1)` перетворює число на рядок і зсуває його, щоб на дисплеї відображалося 1, 2, 3.
- Рядки, що **відступлені** під `for`, знаходяться всередині циклу. Рядки на тому ж рівні, що й `for`, виконуються один раз після завершення циклу.

## Частина 4 – Керування кнопками (Додатково)

На EV3 brick є фізичні кнопки, які можна зчитувати в коді. Це дозволяє створювати прості інтерактивні програми.

```python
#!/usr/bin/env pybricks-micropython

from pybricks.hubs import EV3Brick
from pybricks.ev3devices import Motor
from pybricks.parameters import Port, Button
from pybricks.tools import wait

ev3 = EV3Brick()
motor = Motor(Port.B)

ev3.screen.clear()
ev3.screen.print("Right = Forward")
ev3.screen.print("Left  = Backward")
ev3.screen.print("Center = Quit")
ev3.speaker.beep()

while True:
    pressed = ev3.buttons.pressed()

    if Button.CENTER in pressed:
        break

    elif Button.RIGHT in pressed:
        motor.run(360)

    elif Button.LEFT in pressed:
        motor.run(-360)

    else:
        motor.stop()

    wait(10)

ev3.screen.print("Stopped.")
motor.stop()
```

**Ключові ідеї:**
- `while True:` створює цикл, що виконується нескінченно — поки не буде викликано `break`.
- `ev3.buttons.pressed()` повертає список кнопок, які зараз натиснуті.
- `motor.run()` використовується тут замість `run_time()`, тому що ми хочемо, щоб мотор продовжував обертатися, поки утримується кнопка. `run_time()` заблокує і не дасть нам перевіряти кнопки.
- `wait(10)` дає brick маленьку паузу на кожній ітерації, щоб він не перевантажувався.

## Завдання

Спробуйте самостійно. Кожне будується на тому, що ви вже навчились.

### Завдання 1 – Вперед і назад

Змусіть мотор обертатися **вперед на 720 градусів**, зробіть паузу на 1 секунду, потім обертайтеся **назад на 720 градусів**. Повторіть це **4 рази** за допомогою циклу `for`. Друкуйте номер повторення на екрані щоразу.

### Завдання 2 – Розгін

Змусіть мотор обертатися з **зростаючою швидкістю**: 100, 200, 300, 400, 500 градусів за секунду. Для кожної швидкості запускайте мотор на 1 секунду, потім робіть паузу на пів секунди. Друкуйте поточну швидкість на екрані.

> Підказка: Можна перебирати список значень: `for speed in [100, 200, 300, 400, 500]:`

### Завдання 3 – Абсолютні позиції

Використайте `run_target()`, щоб перемістити мотор у такі позиції по черзі: **0 → 90 → 180 → 270 → 360 → 0**. Додайте паузу в 1 секунду між кожним рухом. Що відбувається, коли ви переходите з 360 назад до 0?

### Завдання 4 – Керування кнопками (Додатково)

Якщо ви ще не ввели програму керування кнопками з Частини 4, зробіть це зараз. Потім змініть її: зробіть так, щоб **кнопка вгору** запускала мотор швидше (720 гр./с), а **кнопка вниз** — повільніше (180 гр./с). Кнопки вправо/вліво повинні як і раніше керувати напрямком на звичайній швидкості.

## Довідкова картка

| Команда | Що робить |
|---|---|
| `Motor(Port.B)` | Створює об'єкт мотора для порту B |
| `motor.run_time(speed, ms)` | Обертається задану кількість часу, потім зупиняється (блокує) |
| `motor.run_angle(speed, degrees)` | Обертається на задану відстань, потім зупиняється (блокує) |
| `motor.run_target(speed, target)` | Повертається до абсолютної позиції (блокує) |
| `motor.run(speed)` | Починає обертання — НЕ зупиняється самостійно |
| `motor.stop()` | Зупиняє мотор |
| `ev3.buttons.pressed()` | Повертає список кнопок, що натиснуті зараз |
| `while True:` / `break` | Нескінченний цикл / вихід із циклу |
